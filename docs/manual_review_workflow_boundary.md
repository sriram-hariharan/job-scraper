# Manual-Review Workflow Boundary

ApplyLens is an assistive job-search product built around a manual-review
workflow. The app may help the user with:

- discovery
- filtering
- ranking
- read-only previews
- readiness checks
- resume/content guidance
- manual review support

These capabilities may organize evidence, prepare materials, and guide a
human decision. They do not grant application execution or submission
authority.

## Permanent boundary

ApplyLens permits:

- no auto-apply
- no auto-submit
- no autonomous application execution
- no automatic job application submission
- no bypass of manual review

Final job application submission must remain under manual user control. No
agent, provider, API, UI, service, pipeline, feature flag, or future phase may
replace the user's final review and submission decision.

Phase 21 focuses on hardening the manual-review workflow, not enabling autonomous application execution.
